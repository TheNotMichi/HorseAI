## Features

- Modern interface
- Multiple chats
- Multiple AI models
- Streaming responses
- Local execution
- No backend required
- Conversation history
- Custom WebLLM models

---

## Installation

Clone the repository:

```bash
git clone https://github.com/TheNotMichi/HorseAI.git
cd HorseAI
```

Run HorseAI:

```bash
python launch.py
```

---

## Requirements

- Python 3.8+
- WebGPU-compatible browser
