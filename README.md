# Overview

HorseAI is a modern AI chat application that runs large language models directly in your browser using **WebLLM** and **WebGPU**.

Everything runs locally on your device.

* No backend
* No API keys
* No cloud processing

---

# Features

| Feature              | Description                                                    |
| -------------------- | -------------------------------------------------------------- |
| Local AI             | Execute models locally using WebGPU.                           |
| Multi-Chat           | Manage multiple conversations simultaneously.                  |
| Multiple Models      | Switch between different WebLLM models.                        |
| Streaming            | Generate responses in real time.                               |
| Conversation History | Chats are automatically saved.                                 |
| Modern UI            | Clean and responsive interface.                                |
| Custom Models        | Add your own compatible WebLLM models.                         |
| Lightweight Launcher | Starts a local server and opens the application automatically. |

---

# Installation

Clone the repository.

```bash
git clone https://github.com/TheNotMichi/HorseAI.git
cd HorseAI
```

Start HorseAI.

```bash
python launch.py
```

The launcher automatically starts a local server and opens the application in your default web browser.
